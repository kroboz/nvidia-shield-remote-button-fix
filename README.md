# nvidia-shield-remote-button-fix
Guide to replacing the Nvidia Shield Pro remote Netflix button with a blank placeholder and finally avoid annoying accidental presses forever.
- Thanks to corporate agreements where any device pre-loaded with Netflix has to have a Netflix button, the otherwise excellent Shield TV Pro remote has an irritating button. I've cancelled Netflix since it's bad now, but the button on the remote can't be remapped without wonky software hacks. It's a pain. So I removed it.
